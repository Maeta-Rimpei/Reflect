# タスク

## history
- その日の感情と、感情分析結果、ふりかえり内容しか表示されていない
    - 分析内容（要約以外）を表示する
- 編集ダイアログは気持ちもう少し大きくてもいい（縦方向）
- 編集ダイアログでテキストエリアにスクロールバーが表示されたときに左角が欠ける
- テキストエリアがリサイズできないようにする
- ふりかえり編集ダイアログ内でも編集と同時に分析をやり直すかどうかのチェックボックスをつける
    - デフォルトはチェックなし
        - 再分析をする場合はチェックしてほしい旨を書いておく
